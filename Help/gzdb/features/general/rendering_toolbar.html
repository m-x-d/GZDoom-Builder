Rendering toolbar

The Rendering toolbar allows you to toggle several enhanced rendering features:

  1. Allows to select dynamic lights rendering mode used in Visual mode.
  2. Allows to select model rendering mode.
  3. Toggles fog rendering in Visual mode.
  4. Toggles sky rendering in Visual mode.
  5. Toggles Event lines.
  6. Toggles Visual vertices in Visual mode (UDMF only).

Most of these settings can also be toggled using actions found in "Rendering" category or appropriate View menu items.
You can toggle rendering effects using "Toggle Enhanced Effects" action.
