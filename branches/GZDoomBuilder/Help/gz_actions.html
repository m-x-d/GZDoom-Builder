Actions

List of new actions avaliable in GZDoom Builder:

You can bind these actions to keyboard shortcuts in Preferences (F5) -> Controls

Drawing:

Name:
Default shortcut:
 Description:
Start Rectangle Drawing
Ctrl-Shift-D
 Starts drawing rectangle. Increase / Decrease Sudivision Level and Increase / Decrease Corners Bevel actions are avaliable in this mode.
Start Ellipse Drawing
Ctrl-Alt-D
 Starts drawing ellipse. Increase / Decrease Sudivision Level and Increase / Decrease Corners Bevel actions are avaliable in this mode.
Bridge Mode
Ctrl-B
 Select two lines or two series of lines, then activate this tool to draw a bezier path between them. Increase / Decrease Sudivision Level actions are avaliable in this mode.
Increase Sudivision Level
-
 Increases subdivision level in Rectangle and Ellipse Drawing Modes.
Decrease Sudivision Level
-
 Decreases subdivision level in Rectangle and Ellipse Drawing Modes.
Increase Corners Bevel
-
 Increase corners bevel in Rectangle Drawing Modes. Bevel can be negative.
Decrease Corners Bevel
-
 Decreases corners bevel in Rectangle Drawing Modes. Bevel can be negative.

Tools:

Name:
Default shortcut:
 Description:
Reload MODELDEF
-
 Reloads MODELDEF. Useful when resource files have been changed outside of Doom Builder (this action can be also found in Tools menu).
Reload GLDEFS
-
 Reloads GLDEFS. Useful when resource files have been changed outside of Doom Builder (this action can be also found in Tools menu).
Reload (Z)MAPINFO
-
 Reloads (Z)MAPINFO. Useful when resource files have been changed outside of Doom Builder (this action can be also found in Tools menu).
Open Color picker
K
 Select dynamic light thing(s) or sector(s), then use this panel to set light properties quickly.
Open UDMF Controls
Ctrl-T
 Select surface(s) in Visual Modes, then use this panel to edit sector and texture properties quickly.
Test map from current position
Ctrl-F9
 Starts the game and loads this map for playing. Player start is placed either at cursor position (in 2D-modes) or at camera position (in Visual Modes).
