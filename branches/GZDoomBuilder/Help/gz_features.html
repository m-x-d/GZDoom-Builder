GZDoom Builder features

List of new features avaliable in GZDoom Builder:

  • Dynamic lights (all types) are rendered in Visual mode.
  • Dynamic lights defined in GLDEFS are rendered in Visual mode.
  • Animated dynamic lights.
  • MD2 and MD3 models rendering in 2D and 3D modes.
  • Fog rendering in Visual mode (including colored fog in maps in UDMF format).
  • GZDB Visual mode.
  • More complete TEXTURES support.
  • Event lines.
  • Visual vertices.
  • Custom linedef colors.
  • Multiple engines per game configuration.
  • Partial GLDEFS and (Z)MAPINFO support.
  • Enhanced scripting workflow.
  • Enhanced textures browser.
  • Draw Rectangle, Draw Ellipse and Bridge modes.
  • Ability to move, cut, copy, paste, insert and delete Things in Visual mode.
  • "Test Map from current position" feature.
  • "Sync camera position between 2D and 3D modes" feature.
  • "Place Things at cursor position in Visual Mode" feature.
  • PNG image format support.
  • PK7 archive format support.
  • Color Picker plugin.
  • UDMF Controls plugin.
  • Tag Explorer plugin.

"Smaller" features:

  • Map Analysis Mode: analysis results can be saved to a text file.
  • Find and Replace mode now have "Sector Brightness" search mode.
  • UDMF: Vertex height offsets ("zfloor" and "zceiling") are shown in Visual mode.
  • UDMF: vertical offsets of a vertex can be edited in Vertex Edit form and are shown in Vertex Info panel.
  • UDMF: UDMF texture offsets can be edited in Sector and Linedef edit forms.
  • Map Analysis window now have "Apply to all errors of this type" checkbox.
  • Up to 8 newly created sectors can now be highlighted using a custom color.
  • Visual mode: things with zero Height and Radius are now rendered.
  • Visual mode: when the thing category has sprite, and the thing doesn't, category sprite is used.
  • Thing Edit Form now has "Random angle" checkbox.
  • Error Checker checks for unknown things.
  • Size of vertex handles in 2D modes can be changed in preferences.
  • etc., etc., etc...
