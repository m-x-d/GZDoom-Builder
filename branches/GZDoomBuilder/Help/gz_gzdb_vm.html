GZDB Visual mode

GZDB Visual mode is the result of merging Doom Builder's Visual mode and GZDoom Visual mode.

List of new features avaliable in this mode:

  • (UDMF) Vertex offsets can be edited.
  • Transparent 3D floors are rendered correctly.
  • Auto-align textures actions now affect sides of 3D floors.
  • Things with zero Height and Radius are rendered.
  • "16: Use a sidedef's upper texture to draw the sides" flag is supported.
  • "32: Use a sidedef's lower texture to draw the sides" flag is supported.
  • "64: Render the 3D floor using additive translucency" flag is supported.
  • "Ignore bottom height" flag is supported.

Warning: GZDoomEditing.dll plugin is no longer supported by GZDoom Builder and should be removed from "Plugins" folder.
