Tag Explorer plugin

Tag Explorer plugin lets you view all things, sectors and linedefs, which have tags and/or actions/effects set. If current map is in UDMF map format, it also allows you to edit things, sectors and linedefs comments and search an item by a comment.

Usage:
Right-click an item to open appropriate Edit menu.
Double-click an item to edit comment (UDMF-only). Remove comment text to clear comment.
