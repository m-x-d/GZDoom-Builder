GZDoom toolbar

GZDoom toolbar allows you to toggle several GZDoom Builder's features:

  1. Toggles models, dynamic lights and fog rendering in Visual mode.
  2. Toggles dynamic lights rendering in Visual mode.
  3. Toggles dynamic lights animation in Visual mode.
  4. Toggles models rendering in Visual mode.
  5. Render all models / render only models for selected things in Visual mode.
  6. Toggles fog rendering in Visual mode.
  7. Toggles Event lines.
  8. Toggles Visual vertices in Visual mode (UDMF only).
