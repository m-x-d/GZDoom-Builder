GZDoom Builder Actions Reference

Classic Modes
Action
Shortcut
Modifiers
Description
Accept Action
Enter
Accepts the changes in the current action and switches back to normal editing mode.
Cancel Action
Escape
Cancels the current action and switches back to normal editing mode.
Copy Properties
Ctrl+Shift+C
Copies the properties of the highlighted or selected object for pasting.
Edit
RButton
Ctrl, Shift
 Edits the properties of the selected items or drags the selected items. Also initiates drawing or inserts new things when no selection is made. Can be used in combination with Control and Shift for the first drawn vertex.
Fit To Screen
Home
Zooms out the map and centers it to make it completely fit in the screen, giving a high overview of the map.
Grid Setup
Shows the Custom Grid Setup dialog which allows you to set custom grid sizes and a background image.
Paint Selection
MButton
Ctrl, Shift
 Selects or deselects items by dragging the mouse. Hold shift while dragging to toggle additive selection. Hold Ctrl while dragging to enable subtractive selection
Pan View
Space
Pans the map in the direction of the mouse while held down.
Paste Properties
Ctrl+Shift+V
Pastes the copied properties onto the highlighted or selected objects.
Place Visual Mode Camera
Ctrl+W
Places a new, or moves the existing, Visual Mode start thing to the mouse position. This thing will keep track where you left Visual Mode so that you return to Visual Mode in the same location.
Scroll East
Right
Scrolls the 2D map view to the right.
Scroll North
Up
Scrolls the 2D map view up.
Scroll South
Down
Scrolls the 2D map view down.
Scroll West
Left
Scrolls the 2D map view to the left.
Select
LButton
Ctrl, Shift
 Selects the highlighted item. Also allows selection by dragging a rectangle. Hold shift while dragging a rectangle to toggle additive or normal selection. Hold Ctrl to enable subtractive selection. Hold Ctrl-Shift to intersect with current selection.
Tag Range
Allows giving a range of tags to a selection of sectors, linedefs or things.
Zoom In
ScrollUp
Zooms in on the map at the current mouse location.
Zoom Out
ScrollDown
Zooms out on the map from the current mouse location.

Tools
Action
Shortcut
Modifiers
Description
Configure Things Filters
Shows the Things Filters setup dialog which allows you to add, remove and change the things filters.
Export to Wavefront .obj
Exports selected sectors (or the whole map if no sectors selected) to Wavefront .obj
Game Configurations
F6
Shows the Game Configurations dialog which allows you to configure settings such as nodebuilder, testing program and resources.
Open Color Picker
K
Select dynamic light thing(s) or sector(s), then use this panel to set light properties quickly.
Preferences
F5
Shows this Preferences dialog.
Reload (Z)MAPINFO
Ctrl+F7
Reloads (Z)MAPINFO. Useful when resource files have been changed outside of Doom Builder.
Reload GLDEFS
Ctrl+F6
Reloads GLDEFS. Useful when resource files have been changed outside of Doom Builder.
Reload MODELDEF
Ctrl+F5
Reloads MODELDEF. Useful when resource files have been changed outside of Doom Builder.
Reload Resources
F8
Reloads all data resources such as game configuration, textures and flats. Useful when resource files have been changed outside of Doom Builder.
Save Screenshot
F12
Saves a screenshot of editor's window into 'Screenshots' folder.
Save Screenshot (editing area only)
Ctrl+F12
Saves a screenshot of editing area into 'Screenshots' folder.
Show Errors and Warnings
F11
Shows the errors and warnings that may have occurred during loading or editing operations.
Statistics
Switches to the map statistics display.
Test Map
F9
Starts the game and loads this map for playing.
Test map from current position
Ctrl+F9
Starts the game and loads this map for playing. Player start is placed either at cursor position (in Classic modes) or at camera position (in Visual Mode).
Toggle Full Brightness
B
Toggles the use of sector brightness on and off. When sector brightness is off, the world is displayed fully bright, without lighting effects.

Visual Modes
Action
Shortcut
Modifiers
Description
Auto-align Textures X
A
Automatically aligns the neighbouring textures X offsets until another texture is encountered.
Auto-align Textures X and Y
Ctrl+A
Automatically aligns the neighbouring textures X and Y offsets until another texture is encountered.
Auto-align Textures Y
Shift+A
Automatically aligns the neighbouring textures Y offsets until another texture is encountered. The Y alignment only takes the ceiling height for each sidedef into account.
Copy Offsets
Shift+C
Copies the targeted texture offsets for pasting.
Copy Properties
Ctrl+Shift+C
Copies the targeted object properties for pasting.
Copy Texture
Ctrl+C
Copies the targeted texture or flat for pasting.
Decrease Brightness by 8
Ctrl+ScrollDown
Decreases the targeted or selected sector brightness level by 8.
Edit
RButton
Edits the properties of the selected items or drags the selected items.
Fit Texture
Ctrl+Alt+A
Scales texture to match size of selected surface.
Fit Texture's Height
Alt+Shift+A
Scales height of a texture to match height of selected surface.
Fit Texture's Width
Alt+A
Scales width of a texture to match width of selected surface.
Increase Brightness by 8
Ctrl+ScrollUp
Increases the targeted or selected sector brightness level by 8.
Look Through Selection
Y
Places visual camera at the same position as selected thing and rotates it to match thing's angle.
Lower Floor/Ceiling by 1 mp
Shift+ScrollDown
Lowers the targeted or selected floors/ceilings by 1 mp. This also lowers selected or targeted things.
Lower Floor/Ceiling by 8 mp
ScrollDown
Lowers the targeted or selected floors/ceilings by 8 mp. This also lowers selected or targeted things.
Lower Floor/Ceiling to adjacent sector
PgDn
Ctrl
 Lowers the targeted or selected floors/ceilings to match the height of adjacent sector. Hold Ctrl to lower to lowest surface in selection. Also drops selected things to floor.
Move Backward
D
Shift
 Moves the camera backward in 3D Visual Mode. Use in combination with Shift for double the speed.
Move Down
Shift
 Moves the camera down in 3D Visual Mode. Use in combination with Shift for double the speed.
Move Forward
E
Shift
 Moves the camera forward in 3D Visual Mode. Use in combination with Shift for double the speed.
Move Left (strafe)
S
Shift
 Strafes the camera left in 3D Visual Mode. Use in combination with Shift for double the speed.
Move Right (strafe)
F
Shift
 Strafes the camera right in 3D Visual Mode. Use in combination with Shift for double the speed.
Move Texture Down by 1
Down
Moves the offset of the targeted or selected textures down by 1 pixel.
Move Texture Down by 8
Shift+Down
Moves the offset of the targeted or selected textures down by 8 pixels.
Move Texture Left by 1
Left
Moves the offset of the targeted or selected textures to the left by 1 pixel.
Move Texture Left by 8
Shift+Left
Moves the offset of the targeted or selected textures to the left by 8 pixels.
Move Texture Right by 1
Right
Moves the offset of the targeted or selected textures to the right by 1 pixel.
Move Texture Right by 8
Shift+Right
Moves the offset of the targeted or selected textures to the right by 8 pixels.
Move Texture Up by 1
Up
Moves the offset of the targeted or selected textures up by 1 pixel.
Move Texture Up by 8
Shift+Up
Moves the offset of the targeted or selected textures up by 8 pixels.
Move Thing Backward
Down
Moves selected Things closer to camera in Visual Modes.
Move Thing Forward
Up
Moves selected Things away from camera in Visual Modes.
Move Thing Left
Left
Moves selected Things left in Visual Modes.
Move Thing Right
Right
Moves selected Things right in Visual Modes.
Move Thing To Cursor Location
Ctrl+MButton
Moves selected Things to cursor location preserving relative offsets in Visual Modes.
Move Up
Shift
 Moves the camera up in 3D Visual Mode. Use in combination with Shift for double the speed.
Paste Offsets
Shift+V
Pastes the copied texture offsets onto the targeted or selected walls.
Paste Properties
Ctrl+Shift+V
Pastes the copied properties onto the targeted or selected object.
Paste Texture
MButton
Pastes the copied texture onto the targeted or selected walls, floors or ceilings.
Paste Texture Flood-Fill
Shift+MButton
This allows you to flood-fill all adjacent textures or flats that are identical to the original with the copied texture.
Raise Floor/Ceiling by 1 mp
Shift+ScrollUp
Raises the targeted or selected floors/ceilings by 1 mp. This also raises selected or targeted things.
Raise Floor/Ceiling by 8 mp
ScrollUp
Raises the targeted or selected floors/ceilings by 8 mp. This also raises selected or targeted things.
Raise Floor/Ceiling to adjacent sector
PgUp
Ctrl
 Raises the targeted or selected floors/ceilings to match the height of adjacent sector. Hold Ctrl to raise to highest surface in selection. Also aligns selected things to ceiling.
Reset Texture Offsets
Shift+R
Resets the texture offsets on the targeted or selected sidedef to 0, 0. Also resets sidedef's scale and floor/ceiling's scale and rotation in UDMF map format.
Rotate Clockwise
Ctrl+Shift+ScrollUp
Rotates selected things clockwise. Also rotates floor/ceiling textures in UDMF map format.
Rotate Counterclockwise
Ctrl+Shift+ScrollDown
Rotates selected things counterclockwise. Also rotates floor/ceiling textures in UDMF map format.
Scale Texture Down (X)
NumPad4
Decreases horizontal scale of targeted or selected textures by 0.25 (UDMF only).
Scale Texture Down (Y)
NumPad5
Decreases vertical scale of targeted or selected textures by 0.25 (UDMF only).
Scale Texture Up (X)
NumPad6
Increases horizontal scale of targeted or selected textures by 0.25 (UDMF only).
Scale Texture Up (Y)
NumPad8
Increases vertical scale of targeted or selected textures by 0.25 (UDMF only).
Select
LButton
Ctrl, Shift
 Selects the highlighted item. Hold Shift to select adjacent surfaces with the same texture. Hold Ctrl to adjacent surfaces with the same height
Select Texture
Ctrl+RButton
Opens the texture browser to select a texture for the targeted or selected walls, floors or ceilings.
Show Things
T
Cycles through the different ways the things are shown in Visual Mode.
Toggle Gravity
G
Toggles the use of gravity while moving around in visual mode. Turn gravity off to fly around, turn gravity on to walk on the sector floors.
Toggle Geometry Effects
Tab
Toggles rendering of (G)ZDoom geometry effects (slopes, 3D-floors etc.) in Visual mode.
Toggle Highlight
H
Toggles the highlight of the targeted object in visual mode.
Toggle Lower Unpegged
L
Toggles the Lower Unpegged setting on the selected or targeted linedef.
Toggle Slope
Alt+S
Toggles Slope for selected surfaces. Select or highlight upper/lower walls to add a slope. Select or highlight floors or ceilings to remove slope.
Toggle Upper Unpegged
U
Toggles the Upper Unpegged setting on the selected or targeted linedef.
Toggle Visual Vertices
Alt+V
When enabled, shows vertices, which heights can be changed, in Visual mode (UDMF only).
Edit
Action
Shortcut
Modifiers
Description
Clear Selection
C
Deselects all selected elements.
Copy Selection
Ctrl+C
Copies the current selection to the clipboard.
Cut Selection
Ctrl+X
Copies the current selection to the clipboard and removes it from the map.
Delete Item
Del
Deletes the highlighted or selected items, depending on the editing mode you are in.
Flip Selection Horizontally
Flips the selection in Edit Selection mode horizontally.
Flip Selection Vertically
Flips the selection in Edit Selection mode vertically.
Grid Decrease
]
Decreases the grid size, increasing the grid density.
Grid Increase
[
Increases the grid size, decreasing the grid density.
Insert Item
Ins
Creates a new item depending on the editing mode you are in.
Map Options
F2
Shows the Map Options dialog which allows changing the map lump name, game configuration and custom resources.
Merge Geometry
Toggles automatic merging of geometry for vertices and structures that are being dragged.
Paste Selection
Ctrl+V
Pastes the current contents of the clipboard into the map as a new selection.
Paste Selection Special
Allows you to choose options or pasting and then pastes the current contents of the clipboard into the map as a new selection.
Redo
Ctrl+Y
Repeats the action(s) performed before Undo was used.
Set Current Textures
Opens the window, which lets user to change textures, which will be used in next drawing operations.
Snap Selected Map Elements to Grid
Snaps selected map elements to grid.
Snap to Grid
Toggles snapping to the grid for things and vertices that are being dragged.
Undo
Ctrl+Z
Restores the current map as it was before last action(s) performed.
View Used Tags
Opens Tag Statistics form, which shows all tags, which are used in current map, and allows to create and edit tag labels.
File
Action
Shortcut
Modifiers
Description
Close Map
Closes the current map and the WAD file in which it exists.
New Map
Ctrl+N
Starts with a new, empty workspace to begin drawing a map from scratch.
Open Map
Ctrl+O
Opens an existing map from WAD file for viewing or modifying.
Open Map in current WAD
Ctrl+Shift+O
Opens an existing map from already loaded WAD file for viewing or modifying.
Save Map
Ctrl+S
Saves the current map to the opened source WAD file.
Save Map As
Saves the current map and all resources from the source WAD file to a new WAD file.
Save Map Into
Saves the current map without any other resources into an existing or new WAD file.
View
Action
Shortcut
Modifiers
Description
Script Editor
F10
This opens the script editor that allows you to edit any scripts in your map or any script files.
Toggle Event lines
I
When enabled, shows order, in which patrol points and interpolation points are connected. Also shows action target for currently highlighted item in 2D modes.
Toggle Info Panel
~
Toggles the info panel between expanded and collapsed.
View Brightness Levels
This sets the view mode to Brightness Levels. This fills all sectors with their brightness levels.
View Ceiling Textures
This sets the view mode to Ceiling Textures. In this view mode the sectors are filled with their ceiling texture and with their brightness level applied.
View Floor Textures
This sets the view mode to Floor Textures. In this view mode the sectors are filled with their floor texture and with their brightness level applied.
View Wireframe
This sets the view mode to Wireframe. This shows only the map elements without any sector filling.
