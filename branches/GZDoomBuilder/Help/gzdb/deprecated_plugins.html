List of deprecated plugins

The following plugins, which shipped with either Doom Builder 2 or older versions of GZDoom Builder, are no longer required or supported.
They can potentially cause problems in GZDoom Builder 2.3 or newer.
It is recommended to delete them from the "Plugins" folder.

  • GZDoomEditing.dll - was merged into BuilderModes.dll in GZDB 1.14. Will cause crash during startup in the SVN versions of GZDB 1.14.
  • Statistics.dll - merged into the core in the SVN versions of GZDB 1.14. Statistics are now shown in any editing mode when not highlighting any map elements.
  • UDMFControls.dll - plugin's functionality was fully replaced by Edit Linedef and Edit Sector windows in the SVN versions of GZDB 1.14.
  • CopyPasteSectorProps.dll - merged into BuilderModes.dll in the SVN versions of GZDB 1.14.

The following unfinished/example plugins either ship or shipped with GZDB's source code.
None of them are required by the end users and can be safely removed from the "Plugins" folder.

  • ImageDrawingExample.dll - a simple editing mode example plugin. Can be useful for people interested in making their own plugins for DB2 or GZDB.
  • USDF.dll - a very unfinished Universal Strife Dialog Format (zdoom.org/wiki/USDF) plugin. Doesn't do anything useful.
  • WadAuthorMode.dll - another very unfinished plugin, which aimed to replicate WadAuthor's all-in-one editing mode. Doesn't do anything useful.
