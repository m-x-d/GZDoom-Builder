TEXTURES support

GZDoom Builder adds support for the following TEXTURES parameters:

  • FlipX, FlipY, Rotate and Blend (both styles) patch parameters.
  • "Add", "Modulate", "Subtract", "ReverseSubtract" and "Translucent" patch Translucency Styles.

GZDoom Builder supports the following special comments:

  • //$GZDB_SKIP - skip the rest of the current file. Useful if you don't want textures from certain files or parts of files to show up in the Image Browser or want to increase resource loading speed.
